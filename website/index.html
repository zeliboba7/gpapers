Welcome...

This is gPapers, the open-sourced, Gnome based digital library manager. Think of it as an iTunes for your PDFs.

I've written this mostly because my desk looks like this. You read a lot during your PhD, and keeping printouts on your desk in a giant stack just doesn't scale. Necessity is the mother of invention...

Download / Install

No nice packages / installer yet, sorry. You'll need the following libraries:

  • Ubuntu: sudo apt-get install python python-glade2 python-gnome2 python-gconf python-sqlite3
  • Debian: sudo apt-get install python python-glade2 python-gnome2 python-sqlite3
  • Redhat: sudo yum install pygtk2 gnome-python2-gconf pygtk2-libglade python-sqlite3

Note: If your distro doesn't have python-sqlite3, you can use pysqlite2.

Now run:

  1. svn checkout http://gpapers.googlecode.com/svn/trunk/ gpapers
  2. svn co http://code.djangoproject.com/svn/django/trunk/django gpapers/django
  3. svn checkout http://deseb.googlecode.com/svn/trunk/src/deseb gpapers/deseb

Note: The last two lines aren't necessary if you have recent versions of django and deseb already installed.

Updates

From your install directory, run: svn update
